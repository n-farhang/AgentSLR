## Relative contribution: parameter value extraction

The relative contribution represents the proportion of disease transmission attributable to different transmission routes. This is intended for pathogens (e.g. MERS) where there is both human to human (h2h) and animal to human (a2h) transmission, and aims to capture the relative magnitude of these two routes of infections in humans.

Using the tool provided, extract the numerical value in the `value` field and specify the transmission type in the `type` field.

The `type` field can only take one of two values:
- `human_to_human`: for transmission from human to human
- `zoonotic_to_human`: for transmission from animals (zoonotic sources) to humans

Please extract relative contributions as written in the paper.
