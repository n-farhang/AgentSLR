Parameter class: **relative contribution**

This parameter is intended for pathogens (e.g. MERS) where there is both human to human (h2h) and animal to human (a2h) transmission, and aims to capture the relative magnitude of these two routes of infections in humans. We expect these to be proportions or percentages. E.g., a study might estimate 60% of infections in humans to be from h2h infection.
