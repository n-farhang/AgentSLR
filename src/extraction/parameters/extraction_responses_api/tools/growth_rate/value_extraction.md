## Growth rate: parameter value extraction

The epidemic growth rate is a key metric that reflects how quickly the number of infections is changing day by day in a population. It is a time-dependent measure, usually expressed as a percentage or a rate per unit of time (e.g., per day), and is crucial for monitoring the speed and trajectory of an outbreak.

Please extract growth rates from the paper. Populate the `value` field with a numerical value as it is specified in the paper. If the paper provides a percentage value like `33%`, record this value as `0.33`. Populate the `unit` field with one of the provided units according to the tool schema.
