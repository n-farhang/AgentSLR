# Notebooks

This directory contains Jupyter notebooks used to generate figures, tables and summary statistics for the paper.

## Notebooks

| Notebook | Purpose |
|----------|---------|
| `data_stats.ipynb` | Data statistics and summary figures |
| `visualisations.ipynb` | Pipeline visualisation and analysis plots |
| `article_screening_eval.ipynb` | Screening evaluation analysis |
| `data_extraction_eval.ipynb` | Extraction evaluation analysis |
| `human_validation.ipynb` | Human expert validation results |
